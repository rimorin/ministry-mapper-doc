    Ministry Mapper 시작하기¶

    개요¶

    Ministry Mapper는 회중이 전도 구역 업무를 디지털 방식으로 구성하는 데 도움을 주는 웹 기반 구역 관리 시스템입니다. 이 가이드를 통해 빠르게 시작할 수 있습니다.

    Ministry Mapper 접근 방법¶

    옵션 1: 호스팅 서비스 사용 (권장)¶

    Ministry Mapper를 사용하는 가장 쉬운 방법은 저희 호스팅 서비스를 이용하는 것입니다:

    ministry-mapper.com

    1. 방문 ministry-mapper.com
    2. 계정 만들기 가입 페이지 사용
    3. 이메일 인증 받은 편지함으로 전송된 링크 클릭
    4. 관리자에게 연락 역할에 맞는 적절한 권한 얻기

    호스팅 서비스의 장점: - 기술적 설정 불필요 - 자동 업데이트 및 백업 - 더 나은 보안 및 신뢰성 - 유지 관리 부담 없음

    옵션 2: 직접 호스팅 (권장하지 않음)¶

    직접 호스팅은 상당한 기술적 전문 지식과 지속적인 유지 관리가 필요합니다. 그래도 직접 호스팅하려면 자세한 지침을 위해 셀프 호스팅 가이드를 참조하십시오.

    빠른 시작 가이드¶

    전도인의 경우¶

    1. 계정 만들기 - 가입하고 이메일 인증
    2. 권한 얻기 - 회중의 구역 담당자에게 접근 권한 요청
    3. 작업 시작 - 배정 링크를 통해 구역에 접근하고 주소 업데이트

    관리자의 경우¶

    1. 호스팅 선택 - 호스팅 서비스 사용 또는 직접 호스팅
    2. 회중 설정 - 구역 및 설정 구성
    3. 사용자 초대 - 사용자 추가 및 역할 배정
    4. 시스템 유지 - 데이터 정확성 유지 및 개인정보 보호 규정 준수 확인

    사용자 역할 이해¶

    Ministry Mapper에는 세 가지 주요 사용자 역할이 있습니다:

    • 전도인 - 배정 링크를 통해 구역에 접근하고 주소 업데이트
    • 인도자 - 구역 배정, 모든 구역 보기, 메시지 관리
    • 관리자 - 구역, 설정 및 사용자 관리에 대한 전체 제어

    각 역할에 대한 자세한 정보는 사용자 가이드를 참조하십시오.

    일반적인 설정 문제¶

    계정을 만들 수 없음¶

    • 유효한 이메일 주소를 사용하는지 확인
    • 비밀번호가 최소 요건을 충족하는지 확인
    • 도움을 위해 관리자에게 연락

    이메일 인증을 받지 못함¶

    • 스팸/정크 폴더 확인
    • 몇 분 기다리거나 재전송 시도
    • 관리자에게 연락

    구역을 볼 수 없음¶

    • 배정된 접근 권한을 얻으려면 인도자 또는 관리자에게 연락
    • 전도인은 배정 링크를 통해서만 구역에 접근

    주소를 업데이트할 수 없음¶

    • 인터넷 연결 확인
    • 배정 링크가 만료되지 않았는지 확인
    • 새로 고침 후 다시 로그인

    더 많은 문제 해결 도움말은 사용자 가이드를 참조하십시오.

    다음 단계¶

    전도인의 경우: 사용자 가이드를 읽고 전도 활동에 관한 모든 기능과 모범 사례를 알아보십시오.

    관리자의 경우: 설정 요건을 위해 셀프 호스팅 가이드를 검토하고 구역 및 사용자 관리를 위해 사용자 가이드를 확인하십시오.

    도움 받기¶

    • 관리자에게 연락 - 접근 및 구역 관련 문의를 위한 첫 번째 연락처
    • 사용자 가이드 읽기 - 포괄적인 기능 문서
    • GitHub 이슈 - https://github.com/rimorin/ministry-mapper/issues에서 버그 및 기술적 문제 신고

    중요 사항¶

    • 인터넷 필요 - Ministry Mapper는 활성 인터넷 연결이 필요합니다
    • 개인정보 보호 준수 - 현지 데이터 보호법(GDPR, CCPA 등) 준수
    • 교육 - 종이에서 디지털로의 원활한 전환을 위해 사용자 교육 제공
