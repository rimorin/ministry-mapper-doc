    Primeros Pasos con Ministry Mapper¶

    Descripción General¶

    Ministry Mapper es un sistema de gestión de territorios basado en la web que ayuda a las congregaciones a organizar su trabajo de ministerio de campo de manera digital. Esta guía te ayudará a comenzar rápidamente.

    Cómo Acceder a Ministry Mapper¶

    Opción 1: Usar el Servicio Alojado (Recomendado)¶

    La manera más sencilla de usar Ministry Mapper es a través de nuestro servicio alojado:

    ministry-mapper.com

    1. Visita ministry-mapper.com
    2. Crea una cuenta usando la página de registro
    3. Verifica tu correo electrónico haciendo clic en el enlace enviado a tu bandeja de entrada
    4. Contacta a tu administrador para obtener los permisos correspondientes a tu rol

    Beneficios del servicio alojado: - No se requiere configuración técnica - Actualizaciones y copias de seguridad automáticas - Mayor seguridad y confiabilidad - Sin carga de mantenimiento

    Opción 2: Auto-Alojamiento (No Recomendado)¶

    El auto-alojamiento requiere conocimientos técnicos significativos y mantenimiento continuo. Si aún deseas auto-alojarlo, consulta la Guía de Auto-Alojamiento para obtener instrucciones detalladas.

    Guía de Inicio Rápido¶

    Para Publicadores¶

    1. Crea tu Cuenta - Regístrate y verifica tu correo electrónico
    2. Obtén Permisos - Contacta al servidor de territorios de tu congregación para obtener acceso
    3. Comienza a Trabajar - Accede a los territorios mediante enlaces de asignación y actualiza las direcciones

    Para Administradores¶

    1. Elige el Alojamiento - Usa el servicio alojado o auto-aloja
    2. Configura la Congregación - Establece territorios y configuraciones
    3. Invita a Usuarios - Agrega usuarios y asigna roles
    4. Mantén el Sistema - Mantén los datos precisos y asegura el cumplimiento de privacidad

    Comprendiendo los Roles de Usuario¶

    Ministry Mapper tiene tres roles principales de usuario:

    • Publicador - Accede a territorios mediante enlaces de asignación y actualiza direcciones
    • Director - Asigna territorios, ve todos los territorios y gestiona mensajes
    • Administrador - Control total sobre territorios, configuraciones y gestión de usuarios

    Para información detallada sobre cada rol, consulta la Guía de Usuario.

    Problemas Comunes de Configuración¶

    No Se Puede Crear una Cuenta¶

    • Verifica que estás usando una dirección de correo electrónico válida
    • Asegúrate de que la contraseña cumple con los requisitos mínimos
    • Contacta a tu administrador para obtener ayuda

    No Se Recibió el Correo de Verificación¶

    • Revisa la carpeta de spam/correo no deseado
    • Espera unos minutos o intenta reenviar
    • Contacta a tu administrador

    No Se Pueden Ver los Territorios¶

    • Contacta a tu director o administrador para que te asigne acceso
    • Los publicadores solo acceden a territorios mediante enlaces de asignación

    No Se Pueden Actualizar las Direcciones¶

    • Verifica tu conexión a internet
    • Comprueba que el enlace de asignación no ha expirado
    • Intenta actualizar la página e iniciar sesión nuevamente

    Para más ayuda con la solución de problemas, consulta la Guía de Usuario.

    Próximos Pasos¶

    Para Publicadores: Lee la Guía de Usuario para aprender todas las características y las mejores prácticas para el servicio de campo.

    Para Administradores: Revisa la Guía de Auto-Alojamiento para los requisitos de configuración y la Guía de Usuario para gestionar territorios y usuarios.

    Obtener Ayuda¶

    • Contacta a tu Administrador - Primer punto de contacto para preguntas sobre acceso y territorio
    • Lee la Guía de Usuario - Documentación completa de características
    • GitHub Issues - Reporta errores y problemas técnicos en https://github.com/rimorin/ministry-mapper/issues

    Notas Importantes¶

    • Se Requiere Internet - Ministry Mapper requiere conexión activa a internet
    • Cumplimiento de Privacidad - Sigue las leyes locales de protección de datos (RGPD, CCPA, etc.)
    • Capacitación - Proporciona capacitación a los usuarios para una transición fluida del papel al formato digital
