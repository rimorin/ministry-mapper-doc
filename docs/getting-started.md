# Getting Started with Ministry Mapper

## Overview

Ministry Mapper is a web-based territory management system that helps congregations organize their field ministry work digitally. This guide will help you get started quickly.

## How to Access Ministry Mapper

### Option 1: Use the Hosted Service (Recommended)

The easiest way to use Ministry Mapper is through our hosted service:

**[ministry-mapper.com](https://ministry-mapper.com)**

1. **Visit** [ministry-mapper.com](https://ministry-mapper.com)
2. **Create an account** using the sign-up page
3. **Verify your email** by clicking the link sent to your inbox
4. **Contact your administrator** to get proper permissions for your role

**Benefits of the hosted service:**
- No technical setup required
- Automatic updates and backups
- Better security and reliability
- No maintenance burden

### Option 2: Self-Hosting (Not Recommended)

Self-hosting requires significant technical expertise and ongoing maintenance. If you still want to self-host, see the [Self-Hosting Guide](self-hosting.md) for detailed instructions.

## Quick Start Guide

### For Publishers

1. **Create Your Account** - Sign up and verify your email
2. **Get Permissions** - Contact your congregation's territory servant for access
3. **Start Working** - Access territories via assignment links and update addresses

### For Administrators

1. **Choose Hosting** - Use hosted service or self-host
2. **Configure Congregation** - Set up territories and settings
3. **Invite Users** - Add users and assign roles
4. **Maintain System** - Keep data accurate and ensure privacy compliance

## Understanding User Roles

Ministry Mapper has three main user roles:

- **Publisher** - Access territories via assignment links and update addresses
- **Conductor** - Assign territories, view all territories, and manage messages
- **Administrator** - Full control over territories, settings, and user management

For detailed information about each role, see the [User Guide](user-guide.md#understanding-user-roles).

## Common Setup Issues

### Can't Create Account
- Verify you're using a valid email address
- Ensure password meets minimum requirements
- Contact your administrator for help

### Email Verification Not Received
- Check spam/junk folder
- Wait a few minutes or try resending
- Contact your administrator

### Can't See Any Territories
- Contact your conductor or administrator to get assigned access
- Publishers access territories via assignment links only

### Can't Update Addresses
- Check your internet connection
- Verify assignment link hasn't expired
- Try refreshing and logging in again

For more troubleshooting help, see the [User Guide](user-guide.md#troubleshooting-common-issues).
