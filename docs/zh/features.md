# 功能概述

## 简介

Ministry Mapper 是一个全面的数字化区域管理平台，专为宗教团体设计。本页面详细介绍了所有功能和特性。

---

## 🗺️ 区域管理

### 组织地理区域

**创建和管理区域：**
- 每个会众可创建无限数量的区域
- 分配唯一的代码和描述
- 通过拖放排序按优先级组织区域
- 自动跟踪完成百分比
- 重置区域状态以重新开始跟踪
- 轻松在区域之间移动地图

**区域统计：**
- 汇总完成情况跟踪
- 可视化进度指示器
- 实时更新统计数据
- 按区域细分
- 历史记录跟踪

**使用场景：** 将您会众的服务区域划分为可管理的区域，以便有组织地进行外勤服务工作。

## 📊 单位/住户跟踪

### 全面的访问状态管理

**状态类型：**

| 状态 | 描述 | 对进度的影响 |
|--------|-------------|-------------------|
| **未完成** | 初始状态，尚未访问 | 计入未完成 |
| **已完成** | 成功完成 | 计入已完成 |
| **不在家** | 无人在家，可重试 | 达到最大尝试次数后计入 |
| **请勿打扰 (DNC)** | 住户要求不访问 | 不计入统计 |
| **无效** | 单位不存在 | 不计入统计 |

**不在家跟踪：**
- 每个会众可配置最大尝试次数
- 自动计数器递增
- 达到最大尝试次数后视为"已完成"
- 防止无限期重访

**完成逻辑：**
```
完成条件：
  - 状态 = 已完成，或
  - 状态 = 不在家 且 尝试次数 >= 最大尝试次数

进度 % = 已完成 / 可统计单位总数 × 100
```

**住户选项：**
- 可自定义地址分类
- 可统计与不可统计类型
- 新地址的默认选项
- 每个地址可有多个选项
- 影响进度计算

**访问备注：**
- 为每个地址添加详细备注
- 自动时间戳跟踪
- 跟踪更新者
- 备注更改时发送电子邮件通知
- 历史备注查看

## 🔗 分配系统（传道员链接）

### 安全、限时访问

**链接功能：**
- **限时：** 可配置过期时间（默认 24 小时）
- **基于令牌：** 安全的 20 字符访问令牌
- **无需账户：** 传道员仅通过链接访问
- **两种类型：**
  - 普通分配用于常规区域工作
  - 个人分配用于个人传道

**智能分配算法：**

请求新分配时，系统：

1. **主要因素：** 选择活跃分配最少的地图
   - 防止单个地图过载
   - 平衡工作量

2. **次要因素：** 优先选择地理位置较近的地图
   - 使用 GPS 坐标
   - 50 米阈值
   - 减少出行时间

3. **第三因素：** 优先选择完成进度较低的地图
   - 确保区域完成均衡
   - 防止某些区域被忽视

**分配工作流程：**
```
1. 负责人生成分配链接
2. 链接分享给传道员（二维码、邮件、短信）
3. 传道员访问 /map/:assignmentId
4. 系统验证令牌和过期时间
5. 传道员实时更新地址状态
6. 链接自动过期
7. 管理员收到实时更新
```

**分配管理：**
- 查看每个地图的活跃分配
- 查看谁拥有哪个区域
- 跟踪分配历史
- 手动删除分配
- 自动清理过期链接（每 5 分钟）

**核心设置：**

**1. 最大不在家尝试次数**

- 拖放重新排序
- 查看哪些地址具有每个选项

**4. 区域设置**
- 时区配置
- 国家/地区选择
- 影响日期计算和报告
