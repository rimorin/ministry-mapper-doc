    Ministry Mapper க்கு வரவேற்கிறோம்¶

    Ministry Mapper என்றால் என்ன?¶

    Ministry Mapper என்பது ஒரு நவீன, திறந்த மூல இணைய பயன்பாடாகும், இது சபைகள் தங்கள் மண்டல ஊழியப் பகுதிகளை திறம்படவாக நிர்வகிக்க உதவுகிறது. காகித பகுதி சீட்டுகளுக்கு பதிலாக, Ministry Mapper இணையதளத்தில் இணைப்பு உள்ள எந்த சாதனத்திலிருந்தும் மண்டல ஊழியப் பணியை ஒழுங்கமைக்கவும், கண்காணிக்கவும், நிர்வகிக்கவும் ஒரு விரிவான டிஜிட்டல் தீர்வை வழங்குகிறது.

    எங்கள் கதை¶

    நாங்கள் 2022 ஆம் ஆண்டில் Ministry Mapper ஐ எங்கள் உள்ளூர் சபை பகுதிகளை மிகவும் திறம்படவாக நிர்வகிக்க உதவும் ஒரு எளிய திட்டமாக தொடங்கினோம். மேலும் சபைகள் டிஜிட்டல் பகுதி நிர்வாகத்தின் நன்மைகளை கண்டறிந்தவுடன், நாங்கள் தளத்தை மேம்படுத்தி வந்தோம். இன்று, Ministry Mapper சிங்கப்பூர் மற்றும் மலேசியாவிலுள்ள சபைகளால் பயன்படுத்தப்படுகிறது, நேர-ஒத்திசைவு, ஊடாடும் வரைப்படம் மற்றும் பல மொழி ஆதரவு போன்ற அம்சங்களுடன் பதிப்பாளர்கள் தங்கள் ஊழியப் பணியில் ஒழுங்கமைக்க உதவுகிறது.

    Ministry Mapper என்ன செய்கிறது¶

    Ministry Mapper விரிவான பகுதி நிர்வாக திறன்களை வழங்குகிறது:

    முக்கிய அம்சங்கள்¶

    • 🗺️ பகுதி நிர்வாகம்: தானியங்கி நிறைவு கண்காணிப்புடன் புவியியல் பகுதிகளை பகுதிகளாக ஒழுங்கமைக்கவும்
    • 🏢 ஊடாடும் வரைப்படம்: OpenStreetMap ஒருங்கிணைப்பு, திசைகள் மற்றும் இடம் கண்டறிதலுடன் Leaflet அடிப்படையிலான வரைப்படங்கள்
    • 📱 நேரடி கூட்டு பணி: SSE (Server-Sent Events) வழியாக அனைத்து சாதனங்களிலும் நேரடி தரவு ஒத்திசைவு
    • 🌍 பல மொழி ஆதரவு: 8 மொழிகள் (ஆங்கிலம், ஸ்பானிஷ், இந்தோனேசியன், ஜப்பானிய, கொரியன், மலாய், சீனம், தமிழ்)
    • 👥 பங்கு அடிப்படையிலான அணுகல்: நுண்-கட்டுப்படுத்தப்பட்ட அனுமதிகளுடன் பதிப்பாளர், நடத்துனர், நிர்வாகி மற்றும் படிக்க-மட்டும் பங்குகள்
    • 🔗 பணி இணைப்புகள்: பயனர் கணக்குகள் இல்லாமல் பகுதி அணுகலுக்கான பாதுகாப்பான, நேர-வரையறுக்கப்பட்ட பகிர்வு இணைப்புகள்
    • 📊 முன்னேற்ற கண்காணிப்பு: காட்சி குறிகாட்டிகள் மற்றும் புள்ளிவிவரங்களுடன் தானியங்கி நிறைவு நிலை
    • ✉️ மின்னஞ்சல் அறிவிப்புகள்: செய்திகள், குறிப்புகள் மற்றும் மாதாந்திர அறிக்கைகளுக்கான HTML மின்னஞ்சல் வார்ப்புருக்கள்
    • 📈 Excel அறிக்கையிடல்: விரிவான புள்ளிவிவரங்களுடன் மாதாந்திர சபை அறிக்கைகள்
    • 🔐 பல-காரணி அங்கீகாரம்: OTP, MFA மற்றும் OAuth2 (Google) ஆதரவு
    • 🌙 இருண்ட முறை: ஒளி/இருண்ட/கணினி விருப்பங்களுடன் கணினி-விழிப்புணர்வு தீம்மிங்
    • 📲 முற்போக்கான இணைய பயன்பாடு: ஆஃப்லைன் சொத்து சேமிப்பகத்துடன் மொபைல் மற்றும் டெஸ்க்டாப்பில் நிறுவக்கூடியது

    வெவ்வேறு பயனர்களுக்கு¶

    • பகுதி ஊழியர்கள்: விரிவான புள்ளிவிவரங்கள் மற்றும் அறிக்கையிடலுடன் அனைத்து பகுதி பதிவுகளையும் டிஜிட்டல் முறையில் நிர்வகிக்கவும்
    • மண்டல ஊழிய நடத்துனர்கள்: புத்திசாலித்தனமான அல்காரிதங்களுடன் பகுதிகளை விரைவாக நியமிக்கவும் மற்றும் பதிப்பாளர் செயல்பாட்டை கண்காணிக்கவும்
    • பதிப்பாளர்கள்: எளிய இணைப்புகள் வழியாக நியமிக்கப்பட்ட வரைப்படங்களை அணுகவும், முகவரி நிலையை நேரடியாக புதுப்பிக்கவும் மற்றும் திறம்பட கூட்டு பணி செய்யவும்
    • நிர்வாகிகள்: பயனர் நிர்வாகம், சபை அமைப்புகள் மற்றும் தணிக்கை தடங்களுடன் முழு கணினி கட்டுப்பாடு

    Ministry Mapper தீர்க்கும் சிக்கல்கள்¶

    காகித இல்லாமல் போவது¶

    பாரம்பரிய காகித பகுதி சீட்டுகள் தொலைந்துவிடும், சேதமடையும் அல்லது காலப்போக்கில் படிக்க கடினமாகிவிடும். Ministry Mapper எல்லாவற்றையும் டிஜிட்டல் முறையில் மற்றும் கிளவுட்டில் பாதுகாப்பாக வைத்திருக்கிறது. உங்கள் தொலைபேசி, டேப்லெட் அல்லது கணினியிலிருந்து எந்த நேரத்திலும் உங்கள் பகுதிகளை அணுகலாம்.

    சிறந்த ஒழுங்கமைப்பு¶

    பகுதிகள் பல பதிப்பாளர்களிடையே பகிரப்படும்போது, தற்செயலாக அதே முகவரிகளில் பணிசெய்வது எளிதாகிவிடும். Ministry Mapper எல்லாரும் நேரடியாக என்ன நடக்கிறது என்பதை பார்க்க உதவுகிறது, எனவே அதே இடத்தை மூடி வீணாக நேரத்தை செலவிடுவதில்லை.

    உடனடி புதுப்பிப்புகள்¶

    யாரோ ஒரு பகுதியை புதுப்பித்தால், எல்லாரும் மாற்றங்களை உடனே பார்க்கிறார்கள். காகிதங்கள் கைமாற்றப்படுவதற்காகவோ அல்லது புதுப்பிப்புகளை பகிர கூட்டங்களுக்காகவோ காத்திருக்க தேவையில்லை.

    எளிதான பதிவு வைத்தல்¶

    காகித சீட்டுகளில் ஒவ்வொரு மாற்றத்தையும் கைமுறையாக எழுதுவதற்கு பதிலாக, Ministry Mapper எல்லாவற்றையும் தானியங்கியாக கண்காணிக்கிறது. இது மணிக்கணக்கான வேலையை சேமிக்கிறது மற்றும் தவறுகளை தடுக்கிறது.

    Ministry Mapper எவ்வாறு செயல்படுகிறது¶

    நவீன தொழில்நுட்ப அடுக்கு¶

    Ministry Mapper சிறந்த செயல்திறனுக்கு அதிநவீன இணைய தொழில்நுட்பங்களை பயன்படுத்துகிறது:

    முன்-முனை (ministry-mapper-v2):

    • React 19.2.4: தானியங்கி கம்பைலர் மேம்படுத்தலுடன் சமீபத்திய UI நூலகம்
    • TypeScript 5.9.3: குறைந்த பிழைகளுக்கு வகை-பாதுகாப்பான மேம்பாடு
    • Vite 8.0.2: மின்னல்-வேக உருவாக்க கருவி மற்றும் dev சேவையகம்
    • Leaflet 1.9.4: OpenStreetMap உடன் ஊடாடும் வரைப்படம் (API வரம்புகள் இல்லை)
    • Bootstrap 5.3.8: நவீன, பதிலளிக்கும் UI கட்டமைப்பு
    • Wouter 3.9.0: இலகுரக திசைப்பயணம் (3KB vs 40KB மாற்றுகள்)

    பின்-முனை (ministry-mapper-be):

    • Go 1.25.0: உயர்-செயல்திறன் கம்பைல்டு மொழி
    • PocketBase 0.36.7: SQLite உடன் சுய-நேர பின்-முனை-சேவையாக
    • SQLite 1.40.2+: நம்பகமான, சுய-இயங்கும் தரவுத்தளம் (தூய Go செயலாக்கம்)
    • Echo v5: குறைவான, உயர்-செயல்திறன் இணைய கட்டமைப்பு
    • Sentry 0.43.0: நேரடி பிழை கண்காணிப்பு மற்றும் கண்காணிப்பு
    • MailerSend 1.6.3: பரிவர்த்தனை மின்னஞ்சல் சேவை
    • LaunchDarkly 7.14.6: அம்சக் கொடி நிர்வாகம்
    • Excelize 2.10.1: Excel அறிக்கை உருவாக்கம்
    • OpenAI GPT: அறிக்கைகள் மற்றும் செய்தி சுருக்கங்களுக்கான AI-இயக்கப்படும் சுருக்கங்கள்

    கட்டமைப்பு சிறப்பு அம்சங்கள்¶

    • சுய-நேர்: எந்த விற்பனையாளர் பூட்டு இல்லை, உங்கள் தரவு மீது முழுக் கட்டுப்பாடு
    • பல-குத்தகை: பாதுகாப்பிற்காக சபை-நிலை தரவு தனிமைப்படுத்தல்
    • நேரடி ஒத்திசைவு: அனைத்து கிளையன்ட்களிலும் SSE-அடிப்படையிலான நேரடி புதுப்பிப்புகள்
    • முற்போக்கான இணைய பயன்பாடு: ஆஃப்லைன் சொத்து ஆதரவுடன் அனைத்து சாதனங்களிலும் நிறுவக்கூடியது
    • Docker கண்டெய்னரைசேஷன்: எளிதான வரிசைப்படுத்தல் மற்றும் அளவிடல்
    • API-முதல் வடிவமைப்பு: விரிவான PocketBase SDK ஆதரவுடன் RESTful API
    • AI-இயக்கப்படும் நுண்ணறிவுகள்: புத்திசாலி அறிக்கை சுருக்கங்கள் மற்றும் செய்தி சுருக்கங்களுக்கான விருப்பமான OpenAI GPT ஒருங்கிணைப்பு

    முக்கியமான கருத்தில் கொள்ள வேண்டியவை¶

    முதலில் தனியுரிமை¶

    Ministry Mapper குடியிருப்பு முகவரிகளை கண்காணிக்கிறது, அதாவது உங்கள் பகுதியில் தனியுரிமை சட்டங்களை பின்பற்ற வேண்டும். வெவ்வேறு நாடுகளில் வெவ்வேறு விதிகள் உள்ளன (ஐரோப்பாவில் GDPR அல்லது கலிபோர்னியாவில் CCPA போல). Ministry Mapper ஐ பயன்படுத்துவதற்கு முன்பு உங்கள் உள்ளூர் ஒழுங்குமுறைகளை புரிந்துகொண்டு பின்பற்றுவதை உறுதிசெய்யுங்கள்.

    இணையம் தேவை¶

    Ministry Mapper இணையத்தில் வேலை செய்கிறது, எனவே அதை பயன்படுத்த இணைப்பு தேவைப்படும். உங்கள் பகுதியில் நம்பகமற்ற இணையம் இருந்தால், இது சவால்களை ஏற்படுத்தலாம். WiFi கிடைக்காத நிலையில் மொபைல் தரவில் பயன்பாடு நன்றாக செயல்படுகிறது.

    கற்றல் வளைவு¶

    காகிதத்திலிருந்து டிஜிட்டலுக்கு மாறுவது ஒரு மாற்றம், குறிப்பாக தொழில்நுட்பத்துடன் குறைவாக வசதியுள்ளவர்களுக்கு. உங்கள் சபையில் உள்ள அனைவருக்கும் மாற்றத்தை எளிதாக்க உதவி மற்றும் ஆதரவை நாங்கள் வழங்குகிறோம்.

    தொடங்குவது எப்படி¶

    பயனர்களுக்கு¶

    Ministry Mapper ஐ பயன்படுத்த எளிதான வழி எங்கள் நேர சேவை மூலம்:

    ministry-mapper.com

    எளிமையாக:

    1. ministry-mapper.com ஐ பார்வையிடவும்
    2. பதிவு பக்கத்தைப் பயன்படுத்தி ஒரு கணக்கை உருவாக்கவும்
    3. உங்கள் மின்னஞ்சல் முகவரியை சரிபார்க்கவும்
    4. சரியான அனுமதிகளுக்கு உங்கள் சபையின் நிர்வாகியை தொடர்புகொள்ளவும்
