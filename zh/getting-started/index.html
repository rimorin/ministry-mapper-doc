    Ministry Mapper 快速入门¶

    概述¶

    Ministry Mapper 是一款基于 Web 的地区管理系统，帮助会众以数字方式组织外勤服务工作。本指南将帮助您快速上手。

    如何访问 Ministry Mapper¶

    方式一：使用托管服务（推荐）¶

    使用 Ministry Mapper 最简单的方式是通过我们的托管服务：

    ministry-mapper.com

    1. 访问 ministry-mapper.com
    2. 创建账户，使用注册页面
    3. 验证您的电子邮件，点击发送到您收件箱的链接
    4. 联系您的管理员，获取您角色的适当权限

    托管服务的优势： - 无需技术设置 - 自动更新和备份 - 更好的安全性和可靠性 - 无维护负担

    方式二：自托管（不推荐）¶

    自托管需要大量技术专业知识和持续维护。如果您仍想自托管，请参阅自托管指南获取详细说明。

    快速入门指南¶

    对于传道员¶

    1. 创建账户 - 注册并验证您的电子邮件
    2. 获取权限 - 联系您会众的地区仆人获取访问权限
    3. 开始工作 - 通过分配链接访问地区并更新地址

    对于管理员¶

    1. 选择托管方式 - 使用托管服务或自托管
    2. 配置会众 - 设置地区和相关设置
    3. 邀请用户 - 添加用户并分配角色
    4. 维护系统 - 保持数据准确并确保隐私合规

    了解用户角色¶

    Ministry Mapper 有三种主要用户角色：

    • 传道员 - 通过分配链接访问地区并更新地址
    • 组长 - 分配地区，查看所有地区，管理消息
    • 管理员 - 完全控制地区、设置和用户管理

    有关每个角色的详细信息，请参阅用户指南。

    常见设置问题¶

    无法创建账户¶

    • 确认您使用的是有效的电子邮件地址
    • 确保密码符合最低要求
    • 联系您的管理员寻求帮助

    未收到电子邮件验证¶

    • 检查垃圾邮件/垃圾文件夹
    • 等待几分钟或尝试重新发送
    • 联系您的管理员

    看不到任何地区¶

    • 联系您的组长或管理员以获得分配访问权限
    • 传道员只能通过分配链接访问地区

    无法更新地址¶

    • 检查您的网络连接
    • 确认分配链接是否已过期
    • 尝试刷新页面并重新登录

    如需更多故障排除帮助，请参阅用户指南。

    下一步¶

    对于传道员：阅读用户指南，了解外勤服务的所有功能和最佳实践。

    对于管理员：查阅自托管指南了解设置要求，以及用户指南了解如何管理地区和用户。

    获取帮助¶

    • 联系您的管理员 - 访问和地区问题的第一联系人
    • 阅读用户指南 - 全面的功能文档
    • GitHub Issues - 在 https://github.com/rimorin/ministry-mapper/issues 报告错误和技术问题

    重要提示¶

    • 需要网络连接 - Ministry Mapper 需要有效的互联网连接
    • 隐私合规 - 遵守当地数据保护法律（GDPR、CCPA 等）
    • 培训 - 为用户提供培训，以顺利从纸质过渡到数字化
