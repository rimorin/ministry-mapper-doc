    Ministry Mapper 用户指南¶

    简介¶

    欢迎使用 Ministry Mapper，这是一个现代化的 Web 应用程序，旨在帮助会众高效管理外勤服务区域。本指南将带您了解开始使用和充分利用该应用程序所需的一切。

    什么是 Ministry Mapper？

    Ministry Mapper 是一个数字化区域管理系统，取代了传统的纸质方法。它允许会众：

    • 以数字方式组织和分配区域
    • 实时跟踪外勤服务访问
    • 协调多位传道员的活动
    • 从任何有互联网的设备访问区域

    主要优势：

    • ✓ 环保 - 消除纸张浪费
    • ✓ 通过云同步实时更新
    • ✓ 适用于任何设备（桌面、平板、手机）
    • ✓ 集成交互式地图便于导航
    • ✓ 安全的基于角色的访问控制

    入门¶

    创建您的账户¶

    图 1：创建新 Ministry Mapper 账户的注册表单

    第 1 步：访问注册页面

    图 2：带有注册选项的登录页面

    1. 访问您会众的 Ministry Mapper URL
    2. 点击登录页面上的 "注册" 按钮

    第 2 步：选择注册方式

    传统注册 Google OAuth（推荐）
    需要电子邮件、密码（6+ 字符、1 个数字、1 个大写字母）和电子邮件验证 使用 Google 账户一键注册
    需要手动电子邮件验证 自动电子邮件验证
    需要记住另一个密码 无需管理密码
    接受隐私政策和条款 通过 Google 增强安全性

    传统注册： 1. 填写：姓名、电子邮件、密码、确认密码 2. 接受隐私政策和服务条款 3. 点击 "创建账户"

    Google OAuth 注册： 1. 点击"或继续使用"下方的 "使用 Google 登录" 按钮 2. 选择您的 Google 账户 3. 授予 Ministry Mapper 基本个人资料访问权限 4. 账户自动创建

    第 3 步：验证您的电子邮件

    图 3：账户创建后发送的电子邮件验证消息

    1. 检查您的电子邮件收件箱中来自 Ministry Mapper 的验证消息
    2. 点击电子邮件中的验证链接
    3. 您将看到确认您的账户已验证的消息

    第 4 步：等待会众访问权限

    验证后：

    1. 返回登录页面
    2. 使用您的电子邮件和密码登录
    3. 如果启用了一次性密码 (OTP)，请检查您的电子邮件获取代码

    图 4：一次性密码 (OTP) 验证屏幕

    图 5：包含登录验证 OTP 代码的电子邮件

    1. 重要：您还看不到任何区域 - 管理员必须先授予您访问您会众的权限
    2. 联系您会众的区域仆人或管理员请求访问权限

    登录 Ministry Mapper¶

    图 6：用于更快更安全身份验证的 Google OAuth 登录选项

    一旦您的账户经过验证并获得会众访问权限：

    标准登录： 1. 导航到您会众的 Ministry Mapper URL 2. 输入您的电子邮件地址和密码 3. 点击 "登录"

    Google OAuth 登录（更快）： 1. 点击 "使用 Google 登录" 按钮 2. 选择您的 Google 账户 3. 自动登录（无需密码）

    第 3 步：完成 OTP 验证（如已启用）

    如果您的会众启用了一次性密码安全：

    1. 输入凭据后，您将看到 OTP 验证屏幕
    2. 检查您的电子邮件获取验证代码
    3. 输入电子邮件中的 6 位数代码
    4. 点击 "验证" 或 "提交"
    5. 代码在 5-10 分钟后过期

    第 4 步：访问您的仪表板

    • 传道员：您不会看到仪表板 - 使用发送给您的分配链接
    • 只读、负责人、管理员：您将看到适合您角色的仪表板

    💡 提示：在受信任的设备上保持登录状态以方便使用，但在共享计算机上始终注销。

    了解用户角色¶

    Ministry Mapper 使用四级访问控制系统。您的角色决定了您可以访问的功能和可以执行的操作。

    角色层级¶

    管理员（完全访问）
    ↓
负责人（管理分配）
    ↓
只读（仅查看）
    ↓
传道员（仅链接访问）

    👤 传道员¶

    访问方式：通过管理员或负责人发送的分配链接

    传道员可以做什么：

    • ✓ 通过共享链接访问区域
    • ✓ 查看带有交互式地图的区域地图
    • ✓ 访问后更新地址状态
    • 标记为：已完成、不在家、请勿打扰、无效
    • ✓ 为地址添加访问备注
    • ✓ 跟踪"不在家"尝试次数
    • ✓ 查看地址详情和顺序

    传道员不能做什么：

    • ✗ 无仪表板访问权限
    • ✗ 无法查看所有会众区域
    • ✗ 无法创建或管理区域
    • ✗ 无法访问分配链接在设定时间后过期（默认：24 小时，可由会众管理员配置）

    最适合：进行外勤服务的普通传道员

    👓 只读¶

    访问方式：具有只读权限的仪表板登录

    只读用户可以做什么：

    • ✓ 查看会众中的所有区域
    • ✓ 查看完整的地址信息
    • ✓ 查看区域进度和统计数据
    • ✓ 访问区域地图
    • ✓ 查看会众消息

    只读用户不能做什么：

    • ✗ 无法修改任何区域或地址数据
    • ✗ 无法创建或删除区域
    • ✗ 无法管理分配
    • ✗ 无法更改会众设置

    最适合：需要可见性但无编辑能力的监督者

    🎯 负责人¶

    访问方式：具有负责人权限的仪表板登录

    负责人能力包括：

    • ✓ 只读可以做的所有事情，加上：
    • ✓ 创建和管理区域分配
    • ✓ 为传道员生成分配链接
    • ✓ 查看所有分配历史
    • ✓ 访问和发布会众消息
    • ✓ 管理会众选项（住户状态类型）
    • ✓ 查看区域完成状态

    负责人不能做什么：

    • ✗ 无法创建或删除区域
    • ✗ 无法编辑地址详情（地址、单位、楼层）
    • ✗ 无法管理用户角色或权限
    • ✗ 无法修改核心会众设置

    最适合：外勤服务协调员和小组监督

    👑 管理员（区域仆人）¶

    访问方式：具有完全管理权限的仪表板登录

    管理员拥有完全控制权：

    • ✓ 负责人可以做的所有事情，加上：
    • ✓ 创建、编辑和删除区域
    • ✓ 添加、修改和删除地址
    • ✓ 管理建筑物和单位
    • ✓ 配置会众设置
    • 设置最大"不在家"尝试次数
    • 配置链接过期时间
    • 设置住户类型选项
    • ✓ 邀请和管理用户
    • ✓ 分配用户角色和权限
    • ✓ 重置区域和地址
    • ✓ 管理地理定位坐标

    最适合：区域仆人和管理会众区域系统的人员

    💡 注意：如果您需要更改角色或不确定当前角色，请联系您的会众管理员。

    主要功能¶

    仪表板概述¶

    图 6：显示区域选择器和主要控件的管理员仪表板

    仪表板界面根据您的角色而异：

    对于传道员¶

    传道员没有仪表板访问权限。相反，他们：

    • 通过电子邮件或消息接收分配链接
    • 点击链接访问其分配的区域
    • 直接在区域视图中工作
    • 链接在配置的时间后自动过期（默认：24 小时）

    对于负责人和管理员¶

    仪表板提供全面的概述：

    1. 区域选择器（顶部下拉菜单）

    • 选择要查看或管理的区域
    • 显示区域代码和描述
    • 区域之间的快速导航

    2. 主要操作按钮

    • 📋 分配：查看和管理分配链接
    • 💬 消息：发布和查看会众消息
    • ⚙️ 设置：访问会众配置（仅限管理员）
    • 👥 用户：管理用户角色和邀请（仅限管理员）

    3. 区域信息面板

    • 区域代码和描述
    • 显示完成百分比的进度条
    • 最后更新时间戳
    • 总单位数与已完成单位数

    4. 区域视图选项

    • 🗺️ 地图视图：交互式地图显示
    • 📋 列表视图：所有地址的表格显示

    5. 快速拨号菜单（浮动操作按钮）

    图 6a：提供快速访问常用操作的快速拨号浮动操作按钮

    ➕ 按钮（右下角）提供快速访问： - 🗺️ 地图视图模式：切换全屏地图视图 - 📋 快速链接：快速创建分配链接（负责人和管理员） - 基于当前视图的上下文感知操作

    查看区域¶

    传道员区域视图¶

    图 7：显示地图和地址列表的传道员分配区域视图

    访问您的分配：

    1. 点击管理员/负责人发送的分配链接
    2. 区域自动加载，包含：
    3. 显示所有地址的交互式 Google 地图
    4. 每个位置的可点击标记
    5. 要访问的地址/单位列表
    6. 当前进度百分比

    显示的区域信息：

    • 区域代码：标识符（例如，"T-001"）
    • 描述：区域名称或区域
    • 进度条：可视化完成状态
    • 地图：带有地址标记的交互式地图
    • 地址列表：所有地址及其当前状态

    管理员/负责人区域视图¶

    图 8：带有区域选择器和管理选项的负责人仪表板

    查看区域：

    1. 登录到您的仪表板
    2. 从下拉菜单中选择一个区域
    3. 查看详细信息：
    4. 区域代码和描述
    5. 完成统计（例如，"15/20 已完成 - 75%"）
    6. 带有所有位置的交互式地图
    7. 完整的地址/单位列表及详情
    8. 管理按钮（编辑、删除、重置）

    管理选项：

    • ✏️ 编辑区域：更改名称、代码或描述
    • 🗑️ 删除区域：删除整个区域
    • 🔄 重置区域：清除所有地址状态
    • ➕ 添加地址：在区域中创建新地址

    使用地址¶

    了解地址信息¶

    图 9：显示所有地址详情和当前状态的地址卡

    Ministry Mapper 中的每个地址或单位都显示全面的信息：

    基本信息：

    • 地址/单位号：位置标识符（例如，"#05-123"）
    • 楼层：建筑楼层（对于多层物业）
    • 类型：基于会众选项的住户分类
    • 示例：中文、英文、泰米尔语、西班牙语
    • 如果配置，可以选择多种类型
    • 顺序：访问顺序号

    状态信息：

    • 状态：当前访问状态（见下面的状态类型）
    • 不在家次数：未应答访问尝试次数
    • 请勿打扰日期：标记 DNC 的时间（如适用）

    活动跟踪：

    • 备注：关于访问或住户的重要信息
    • 最后更新：最近更新的日期和时间
    • 更新者：进行最后更改的用户名

    地址状态类型¶

    Ministry Mapper 使用五种标准状态类型：

    状态 颜色 描述 用途
    未完成 白色/默认 尚未访问 所有地址的初始状态

    保护信息¶

    Ministry Mapper 处理敏感的地址和个人信息。请遵守这些准则：

    账户安全：

    • ✓ 永远不要分享登录凭据
