    Ministry Mapper をはじめる¶

    概要¶

    Ministry Mapper は、会衆がフィールド奉仕のテリトリーをデジタルで管理するのに役立つウェブベースのテリトリー管理システムです。このガイドでは、すぐに始めるための方法を説明します。

    Ministry Mapper へのアクセス方法¶

    オプション1：ホストサービスを使用する（推奨）¶

    Ministry Mapper を使用する最も簡単な方法は、私たちのホストサービスを通じてです：

    ministry-mapper.com

    1. 訪問 ministry-mapper.com
    2. アカウントを作成 サインアップページを使用して
    3. メールを確認 受信トレイに送られたリンクをクリックして
    4. 管理者に連絡 あなたの役割に適切な権限を取得するために

    ホストサービスのメリット： - 技術的なセットアップ不要 - 自動更新とバックアップ - より良いセキュリティと信頼性 - メンテナンスの負担なし

    オプション2：セルフホスト（非推奨）¶

    セルフホストには、重大な技術的専門知識と継続的なメンテナンスが必要です。それでもセルフホストを希望する場合は、詳細な手順についてセルフホストガイドをご覧ください。

    クイックスタートガイド¶

    伝道者向け¶

    1. アカウントを作成 - サインアップしてメールを確認する
    2. 権限を取得 - 会衆のテリトリー担当者にアクセスを依頼する
    3. 奉仕を開始 - 割り当てリンク経由でテリトリーにアクセスし、住所を更新する

    管理者向け¶

    1. ホスティングを選択 - ホストサービスを使用するかセルフホストする
    2. 会衆を設定 - テリトリーと設定をセットアップする
    3. ユーザーを招待 - ユーザーを追加して役割を割り当てる
    4. システムを維持 - データを正確に保ちプライバシーコンプライアンスを確保する

    ユーザーロールの理解¶

    Ministry Mapper には3つの主要なユーザーロールがあります：

    • 伝道者 - 割り当てリンク経由でテリトリーにアクセスし、住所を更新する
    • コンダクター - テリトリーを割り当て、すべてのテリトリーを表示し、メッセージを管理する
    • 管理者 - テリトリー、設定、ユーザー管理を完全に制御する

    各役割の詳細については、ユーザーガイドをご覧ください。

    よくあるセットアップの問題¶

    アカウントを作成できない¶

    • 有効なメールアドレスを使用していることを確認する
    • パスワードが最小要件を満たしていることを確認する
    • 管理者に連絡してサポートを求める

    メール確認が届かない¶

    • スパム/迷惑メールフォルダを確認する
    • 数分待つか、再送信を試みる
    • 管理者に連絡する

    テリトリーが表示されない¶

    • アクセスを割り当てられた割り当てリンクを取得するために、コンダクターまたは管理者に連絡する
    • 伝道者は割り当てリンク経由のみでテリトリーにアクセスできます

    住所を更新できない¶

    • インターネット接続を確認する
    • 割り当てリンクが期限切れでないことを確認する
    • 更新してログインし直す

    詳細なトラブルシューティングについては、ユーザーガイドをご覧ください。

    次のステップ¶

    伝道者向け：フィールドサービスのすべての機能とベストプラクティスを学ぶためにユーザーガイドをお読みください。

    管理者向け：セットアップ要件についてはセルフホストガイドを確認し、テリトリーとユーザー管理についてはユーザーガイドをお読みください。

    ヘルプの取得¶

    • 管理者に連絡 - アクセスとテリトリーに関する質問の最初の窓口
    • ユーザーガイドを読む - 包括的な機能ドキュメント
    • GitHub Issues - https://github.com/rimorin/ministry-mapper/issues でバグと技術的な問題を報告する

    重要な注意事項¶

    • インターネット必須 - Ministry Mapper はアクティブなインターネット接続が必要です
    • プライバシーコンプライアンス - 地域のデータ保護法（GDPR、CCPAなど）に従ってください
    • トレーニング - 紙からデジタルへの円滑な移行のためにユーザートレーニングを提供してください
