    Ministry Mapper へようこそ¶

    Ministry Mapper とは何ですか？¶

    Ministry Mapper は、会衆がフィールド奉仕のテリトリーを効率的に管理するのに役立つ、最新のオープンソース Webアプリケーションです。紙のテリトリー伝票の代わりに、Ministry Mapper はインターネット接続のある任意のデバイスから、フィールド奉仕活動の整理・追跡・管理のための包括的なデジタルソリューションを提供します。

    私たちのストーリー¶

    私たちは 2022 年に、地元の会衆がテリトリーをより効果的に管理できるよう支援するシンプルなプロジェクトとして Ministry Mapper を始めました。より多くの会衆がデジタルテリトリー管理のメリットを発見するにつれて、プラットフォームを継続的に強化してきました。現在、Ministry Mapper はシンガポールとマレーシアの会衆で使用されており、リアルタイム同期、インタラクティブマッピング、多言語サポートなどの機能で伝道者が奉仕活動を整理するのに役立っています。

    Ministry Mapper でできること¶

    Ministry Mapper は包括的なテリトリー管理機能を提供します：

    コア機能¶

    • 🗺️ テリトリー管理：地理的地域をテリトリーに整理し、自動完了追跡
    • 🏢 インタラクティブマッピング：OpenStreetMap 統合、方向案内、位置情報による Leaflet ベースのマップ
    • 📱 リアルタイムコラボレーション：SSE（Server-Sent Events）経由でのすべてのデバイス間のライブデータ同期
    • 🌍 多言語サポート：8言語（英語、スペイン語、インドネシア語、日本語、韓国語、マレー語、中国語、タミル語）
    • 👥 役割ベースのアクセス：細かな権限を持つ伝道者、コンダクター、管理者、読み取り専用の役割
    • 🔗 割り当てリンク：ユーザーアカウントなしでのテリトリーアクセスのための安全で時間制限付き共有リンク
    • 📊 進捗追跡：視覚的な指標と統計による自動完了ステータス
    • ✉️ メール通知：メッセージ、メモ、月次レポート用の HTML メールテンプレート
    • 📈 Excel レポート：詳細な統計を含む月次会衆レポート
    • 🔐 多要素認証：OTP、MFA、OAuth2（Google）サポート
    • 🌙 ダークモード：ライト/ダーク/システムオプション付きのシステム対応テーマ
    • 📲 プログレッシブWebアプリ：オフラインアセットキャッシュでモバイルとデスクトップにインストール可能

    各ユーザー向け¶

    • テリトリー担当者：包括的な統計とレポートですべてのテリトリー記録をデジタルで管理
    • フィールド奉仕コーディネーター：インテリジェントアルゴリズムでテリトリーを素早く割り当て、伝道者の活動を追跡
    • 伝道者：シンプルなリンク経由で割り当てられたマップにアクセスし、住所ステータスをリアルタイムで更新し、効果的に連携
    • 管理者：ユーザー管理、会衆設定、監査証跡による完全なシステム制御

    Ministry Mapper が解決する問題¶

    ペーパーレス化¶

    従来の紙のテリトリー伝票は紛失したり、損傷したり、時間とともに読みにくくなります。Ministry Mapper はすべてをデジタルでクラウドに安全に保管します。スマートフォン、タブレット、コンピュータからいつでもテリトリーにアクセスできます。

    より良い組織化¶

    複数の伝道者でテリトリーを共有すると、同じ住所を偶然に訪問してしまいやすくなります。Ministry Mapper はすべての人がリアルタイムで状況を把握できるようにし、同じ地域を重複してカバーする無駄をなくします。

    即時更新¶

    誰かがテリトリーを更新すると、すぐに全員が変更を確認できます。更新を共有するために書類を回したり会議を開いたりする必要がありません。

    記録管理の簡素化¶

    紙の伝票にすべての変更を手書きする代わりに、Ministry Mapper がすべてを自動的に追跡します。これにより何時間もの作業が節約され、ミスを防げます。

    Ministry Mapper の仕組み¶

    最新技術スタック¶

    Ministry Mapper は最適なパフォーマンスのために最先端のウェブ技術を活用しています：

    フロントエンド（ministry-mapper-v2）：

    • React 19.2.4：自動コンパイラ最適化を備えた最新の UI ライブラリ
    • TypeScript 5.9.3：バグを減らすための型安全な開発
    • Vite 8.0.2：超高速ビルドツールと開発サーバー
    • Leaflet 1.9.4：OpenStreetMap によるインタラクティブマッピング（API 制限なし）
    • Bootstrap 5.3.8：モダンでレスポンシブな UI フレームワーク
    • Wouter 3.9.0：軽量ルーティング（代替品の 40KB に対して 3KB）

    バックエンド（ministry-mapper-be）：

    • Go 1.25.0：高性能コンパイル言語
    • PocketBase 0.36.7：SQLite 搭載のセルフホスト Backend-as-a-Service
    • SQLite 1.40.2+：信頼性の高い自己完結型データベース（純粋な Go 実装）
    • Echo v5：ミニマリストで高性能な Web フレームワーク
    • Sentry 0.43.0：リアルタイムエラー追跡と監視
    • MailerSend 1.6.3：トランザクションメールサービス
    • LaunchDarkly 7.14.6：フィーチャーフラグ管理
    • Excelize 2.10.1：Excel レポート生成
    • OpenAI GPT：レポートとメッセージダイジェストの AI による要約

    アーキテクチャのハイライト¶

    • セルフホスト：ベンダーロックインなし、データの完全な制御
    • マルチテナント：セキュリティのための会衆レベルのデータ分離
    • リアルタイム同期：すべてのクライアント間での SSE ベースのライブ更新
    • プログレッシブWebアプリ：オフラインアセットサポートですべてのデバイスにインストール可能
    • Docker コンテナ化：簡単なデプロイとスケーリング
    • API ファーストデザイン：包括的な PocketBase SDK サポートを持つ RESTful API
    • AI による洞察：インテリジェントなレポート要約とメッセージダイジェストのためのオプションの OpenAI GPT 統合

    重要な考慮事項¶

    プライバシー最優先¶

    Ministry Mapper は住宅の住所を追跡するため、お住まいの地域のプライバシー法に従う必要があります。国によって異なるルールがあります（ヨーロッパの GDPR やカリフォルニアの CCPA など）。Ministry Mapper を使用する前に、地域の規制を理解し遵守していることを確認してください。

    インターネット必須¶

    Ministry Mapper はインターネット経由で動作するため、使用するには接続が必要です。お住まいの地域でインターネットが不安定な場合、課題が生じる可能性があります。WiFi が利用できない場合でも、モバイルデータ通信でうまく動作します。

    学習曲線¶

    紙からデジタルへの移行は変化であり、特にテクノロジーに不慣れな方には難しいかもしれません。会衆の全員がスムーズに移行できるよう、ヘルプとサポートを提供しています。

    ユーザー向け¶

    Ministry Mapper を使用する最も簡単な方法は、私たちのホストサービスを通じてです：

    ministry-mapper.com

    手順：

    1. ministry-mapper.com にアクセス
    2. サインアップページでアカウントを作成
    3. メールアドレスを確認
    4. 会衆の管理者に適切な権限を依頼

    管理者向け¶

    会衆への Ministry Mapper の導入を検討している場合：

    • 推奨：ministry-mapper.com のホストサービスを使用する

    • 技術的なセットアップ不要

    • プロフェッショナルサポートあり
    • 自動更新とメンテナンス

    • より良い信頼性とセキュリティ

    • 代替：デプロイメントガイドを確認する（上級ユーザーのみ）

    技術的な複雑さと継続的なメンテナンスの要件のため、セルフホストはお勧めしません。ministry-mapper.com のホストサービスはよりシンプルで信頼性の高いソリューションを提供します。
